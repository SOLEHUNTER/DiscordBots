Description: Discord Bot that combines the Discord.js library with the Twitter API.
How to run: 
  1. If you would like to use this bot for yourself make sure you have javascript installed, a discord bot key, and twitter bot keys/secrets.
  2. Edit the config file and add these values.
Discord Commands:
  1. @latest <twitter_name> -> Bot sends the latest tweet from twitter_name in the chat.
  2. @raw <twitter_name> <search_arguments> -> Bot allows you to query the Twitter API using your own search arguments, uses twitter's advanced search system.
  3. @debug (NOT Complete) Used for debugging.
